SM Testimonials is a Testimonial Slider Plugin Use for Display Testimonial Slider Using Shortcode

== Description ==

SM Testimonials is a Testimonial Slider Plugin Use for Display


Primary Features
*  Very easy to use just copy this shortcode [sm-testimonials-slider]
*  A front-end Slider
*  Custom fields
*  Featured Images (thumbnails)
*  Developer-friendly (actions, filters, templates)
*  This plugin is free


== Installation ==

Go to Plugins > Add New.
Search for “sm testimonials”.
Click “Install Now”.
OR

Download the zip file.
Upload the zip file via Plugins > Add New > Upload.
Activate the plugin. Look for “Testimonials” in the admin menu.

== Frequently Asked Questions ==

= What is the shortcode? =

use this shortcode:

[sm-testimonials-slider]
